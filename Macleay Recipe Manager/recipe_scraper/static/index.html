📥 Import Recipes

Add recipes to your current cookbook from a file. All imported recipes go straight into My Cookbook.

📄 Text File (.txt)
A single recipe written in plain text. The first line becomes the title. Label your sections with Ingredients: and Instructions: (one item per line). You can also add Servings:, Category:, and Time: lines.
📕 PDF Recipe (.pdf)
Import a single recipe from a PDF file. The app will extract the text and create a recipe — works best with digital PDFs (not scanned images). You can review and edit the result before saving.
📚 Cookbook File (.cookbook)
A Macleay Recipe Manager cookbook exported from this app or another copy. Every recipe in the cookbook will be imported into your current cookbook.
📊 CSV Spreadsheet (.csv)
Supports two formats: Macleay Recipe Manager CSV (exported from this app — full fidelity, preserves photos and all fields) and AccuChef CSV (exported from AccuChef — imports title, category, servings, ingredients, and instructions). All recipes in the file will be imported.
